# Project Overview

This project is developed for CSC 770. The class project involves a system with major parts including:

- Web Server
- Workbook / VBA side
- Solver Server
- Risk Management / CI support

The general workflow discussed in class is:

1. Data is prepared through workbook/data tabs
2. Information is sent to the solver server
3. The solver server processes the input
4. Results are returned back to the workbook/system
5. Users review results and continue workflow

This repository will document my contribution to the project.
